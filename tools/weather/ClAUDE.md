# Weather Tool
This repository is an MCP server that can be used to fetch the weather from OpenWeatherAPI

## Development guidelines
When creating any elements of this project you must ALWAYS first consult the `best_practices` tool.

For example:
- If asked to implement some code to call an API, you should first consult the best practices mcp server
for `http`.
- When asked to implement a feature for Kubernetes deployment, you should first consult the best practices mcp server
for `kubernetes`.
